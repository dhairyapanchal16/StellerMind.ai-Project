## ⚙️ Deployment Steps

1. Login to Azure Portal  
2. Create a Resource Group  
3. Create an Azure App Service (PHP runtime)  
4. Upload the project using ZIP deployment or GitHub  
5. Verify files in `/site/wwwroot`  
6. Access the deployed website
